### Parking Lot Console App

A simple parking lot management console application written in C++ using classes.

##### Clone Project and Run
first you should clone repository.

```bash
git clone https://github.com/eraycetinx/cpp-car-parking.git
cd cpp-car-parking
```

Then compile the project:


```bash
g++ *.cpp -o parking
```

Run the program:

```bash
./parking
```
